# WASaver-Android

Android app for saving WhatsApp statuses and organizing related media utilities.

## Features

- Save WhatsApp and WhatsApp Business statuses
- Browse and manage saved media
- Direct chat utility
- Release-check integration

## Getting the app

Download releases from:
https://github.com/LeoAristocrat/WASaver-Android/releases

## Development notes

This repo is Android/Gradle based. Open it in Android Studio for builds and debugging.
